## About

This is my personal resume website, featuring a clean and professional design that highlights my work experience, technical skills, education, and contact information. Built with modern web technologies and optimized for all devices.

## Features

- **Responsive Design** - Looks great on desktop, tablet, and mobile devices
- **Smooth Animations** - Fade-in effects and smooth scrolling for better user experience
- **Modern UI** - Clean, professional design with attention to detail
- **Interactive Elements** - Hover effects, parallax scrolling, and animated transitions
- **Accessible** - Semantic HTML and proper contrast ratios
- **Fast Loading** - Vanilla JavaScript with no dependencies for optimal performance

## Sections

- **Hero** - Introduction with name, title, and location
- **Technical Skills** - Organized by category (Backend, Frontend, DevOps, Testing)
- **Work Experience** - Timeline layout with detailed achievements
- **Education** - Academic and self-taught learning background
- **Contact** - Easy access to email, phone, LinkedIn, and GitHub

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Google Fonts (Inter)

## Local Development

To run this website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/fedesapuppo/resume-website.git
   ```

2. Open the `index.html` file in your browser:
   ```bash
   cd resume-website
   open index.html
   ```

That's it! No build process or dependencies required.
